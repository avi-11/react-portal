## Instructions to run

-   Pre-requisites:

    -   Node.js
    -   NPM

-   Directions to setup/install

```bash
$ git clone https://github.com/dsckiet/react-portal
$ cd react-portal
$ npm i
```

-   Directions to execute

```bash
$ npm start
```
